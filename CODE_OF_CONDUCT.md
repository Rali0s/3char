# Code of Conduct

## Our Standard

This project expects respectful, constructive, and professional collaboration.

## Unacceptable Behavior

- Harassment, discrimination, or personal attacks
- Trolling or intentionally disruptive behavior
- Publishing private information without permission

## Enforcement

Project maintainers may remove or reject contributions that violate this policy.
